1.What is DOM and what is the purpose of DOM?

DOM : DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document.It is a method to access all the HTML elements as object.It also defines the events for all HTML elements.The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.
The main purpose of DOM is to create, change, or remove elements from a website document.It transforms a full HTML file as object as like as a model of HTML file of JavaScript.

3.What is event Bubble?

Event Bubbling is a concept in the Document Object Model(DOM).It works when an element receives an event, and that event bubbles up target element to top element or to its parents in the DOM tree until it gets the root element.
